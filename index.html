Introduction to git

Basic commands

  • git init:Initialize a repository
  • git status:Verify the status of the archives
  • git add file_name:Saves the archive in the staging area
  • git commit -m "message":Creates a version of our app
  • git log:Shows the versions of the app

Branching

  • git branch:Shows the branches of your app and where u are located
  • git branch new_branch_name:Adds a new branch
  • git checkout branch_name:Lets you move between branches
  • git branch_name -d:Deletes the branch that you selected
  • git merge branch_name:Merges two branches, the one u are in, and the one u selected
